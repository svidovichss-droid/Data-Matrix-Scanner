ТРЕБОВАНИЯ
───────────
• Windows 10 / 11 (64-бит)
• Python 3.10 или новее — скачать: https://python.org/downloads
  (при установке отметьте "Add Python to PATH")
• Интернет для первоначальной загрузки библиотек


СБОРКА EXE (шаг за шагом)
──────────────────────────
1. Скопируйте на компьютер папку python-exe целиком

2. Дважды щёлкните файл:
       build_exe.bat

   Скрипт автоматически:
   ✓ Установит нужные библиотеки (opencv, pylibdmtx, pillow)
   ✓ Установит PyInstaller
   ✓ Соберёт EXE-файл

3. Готовый файл появится по пути:
       python-exe\dist\DataMatrixScanner.exe

4. Перенесите DataMatrixScanner.exe куда удобно — он работает
   как обычная программа, без установки Python.


ВОЗМОЖНЫЕ ПРОБЛЕМЫ
──────────────────
• «Python не найден» — переустановите Python, отметив Add to PATH
• «Ошибка DLL» при запуске EXE — запустите build_exe.bat снова,
  иногда нужна повторная сборка
• Антивирус блокирует EXE — добавьте папку dist\ в исключения
  (PyInstaller-файлы часто ложно срабатывают)


РАБОТА ПРОГРАММЫ
────────────────
• Нажмите «Старт» — программа откроет камеру
• Поднесите DataMatrix к камере — захват произойдёт автоматически
• Оценка A-F отобразится с цветовой индикацией и звуком:
    A, B  — один двойной сигнал (хорошее качество)
    C     — два сигнала (удовлетворительно)
    D     — чередующаяся сирена (плохо)
    F     — аварийный сигнал (неудовлетворительно)
• Параметры качества по ГОСТ Р 57302-2016:
    SC  — Контраст символа            (п. 5.4)
    MOD — Модуляция                   (п. 5.5)
    RM  — Запас отражательной способности (п. 5.6)
    FPD — Повреждение фикс. рисунка   (п. 5.7)
    ANU — Осевая неравномерность      (п. 5.8)
    GNU — Неравномерность сетки       (п. 5.9)
    UEC — Неиспользованная коррекция  (п. 5.10)
    PG  — Прирост печати              (п. 5.11)
