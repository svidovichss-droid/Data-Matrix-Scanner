# DataMatrix Quality Scanner

**ГОСТ Р 57302-2016 / ISO/IEC 15415**

Авторы: Александр Свидович, Алексей Петляков

---

## О программе

Приложение для автоматической оценки качества печати символов DataMatrix непосредственно с камеры в режиме реального времени. Оценка выполняется по параметрам российского стандарта **ГОСТ Р 57302-2016**, гармонизированного с международным стандартом **ISO/IEC 15415:2011**.

Доступно в двух форматах:
- **Веб-приложение** — работает в браузере Chrome/Edge без установки
- **EXE для Windows** — собирается из Python-скрипта через PyInstaller

---

## Возможности

### Сканирование
- Автоматический захват DataMatrix с камеры без нажатия кнопок
- Поддержка скоростных камер высокого разрешения (Full HD / 4K, до 60+ fps)
- Автоматический выбор наиболее качественной камеры из доступных
- Выбор камеры вручную из выпадающего списка
- Визуальный прицел с анимацией сканирующей линии
- Счётчик частоты кадров (fps) в режиме реального времени

### Оценка качества по ГОСТ Р 57302-2016

Анализируются 8 параметров с выставлением оценки A–F по каждому:

| Параметр | Расшифровка | Пункт ГОСТ |
|----------|-------------|------------|
| SC  | Контраст символа | п. 5.4 |
| MOD | Модуляция | п. 5.5 |
| RM  | Запас отражательной способности | п. 5.6 |
| FPD | Повреждение фиксированного рисунка | п. 5.7 |
| ANU | Осевая неравномерность | п. 5.8 |
| GNU | Неравномерность сетки | п. 5.9 |
| UEC | Неиспользованная коррекция ошибок Рида-Соломона | п. 5.10 |
| PG  | Прирост печати | п. 5.11 |

Итоговая оценка **A / B / C / D / F** определяется по наихудшему параметру — в соответствии с правилом п. 5.3 ГОСТ Р 57302-2016.

### Звуковое сопровождение

| Оценка | Сигнал |
|--------|--------|
| A, B | Один двойной звуковой сигнал |
| C | Два последовательных сигнала |
| D | Чередующаяся сирена (4 тона) |
| F | Аварийная сирена (8 тонов) |

Звук отключается кнопкой в интерфейсе.

### Интерфейс
- Тёмная тема, оптимизированная для производственного освещения
- Цветовая индикация оценки (зелёный → жёлтый → красный)
- Прогресс-бар по каждому параметру с числовым значением
- История последних 50 сканирований с временными метками
- Отображение декодированных данных символа
- Время выполнения анализа в миллисекундах

## Ограничения

### Технические
- **Только DataMatrix** — другие типы штрихкодов (QR, Code128, EAN и др.) не поддерживаются
- **Одиночное сканирование** — одновременное считывание нескольких символов в кадре не предусмотрено; декодируется первый найденный символ
- **Анализ на основе изображения** — параметры качества вычисляются по яркостным характеристикам кадра с камеры, а не по эталонному оптическому измерителю. Результаты являются относительными и не заменяют аттестованное метрологическое оборудование по ГОСТ Р 57302-2016
- **Нет калибровки** — отсутствует калибровка по эталонному символу или источнику освещения, что влияет на абсолютную точность параметров SC и RM

### Требования к условиям съёмки
- Освещение должно быть равномерным; блики, тени и переотражения снижают достоверность SC, MOD, RM
- Оптимальное расстояние: символ должен занимать не менее 30% ширины кадра
- Сильное смазывание (motion blur) снижает точность FPD и GNU
- Перспективные искажения при угле съёмки более 20° ухудшают ANU и GNU

### Веб-версия
- Работает в браузерах с поддержкой WebRTC: Chrome 80+, Edge 80+, Firefox 75+
- Требует разрешения браузера на доступ к камере
- Звук воспроизводится только после первого взаимодействия со страницей (ограничение браузеров)
- Safari на iOS имеет ограниченную поддержку камеры через WebRTC

### EXE-версия (Windows)
- Только Windows 10 / 11 (64-бит)
- Для сборки требуется Python 3.10+ с установленным PATH
- Антивирусы могут ложно срабатывать на файлы PyInstaller — требуется добавление в исключения

## Сборка EXE

1. Установить Python 3.10+ с [python.org](https://python.org) (отметить **Add Python to PATH**)
2. Перенести папку `python-exe` на компьютер с Windows
3. Запустить `build_exe.bat` двойным щелчком
4. Готовый файл: `python-exe\dist\DataMatrixScanner.exe`

## Деплой веб-версии
